# 경기도 응급의료 자원 분포 대시보드

경기도 응급의료기관의 **지역별 분포와 형평성**을 한눈에 보여주어 **응급의료 사각지대**(자원 부족 지역)를 드러내는 대시보드입니다.

**🔗 라이브 데모**: https://anywaysoul92.github.io/gyeonggi-emergency-dashboard/

---

## 한 줄 인사이트
> 경기도 31개 시군 중 **6곳(하남·과천·의왕·가평·동두천·양주)에는 응급의료기관이 한 곳도 없고**, 인구 10만 명당 기관 수는 지역 간 **약 10배(0.23 → 2.38)** 까지 벌어진다.

## 누가 보는가 (독자)
- 1차: 경기도·시군 응급의료 정책 담당자, 응급의료지원센터 기획자 — 사각지대 식별과 신규 지정·예산 근거
- 2차: 소방·구급 상황실, 지역 보건의료 연구자 — 자원 분포 파악

## 핵심 KPI (상단 카드)
| KPI | 값 | 의미 |
|---|---|---|
| 총 응급의료기관 | 72개 | 경기도 전체 자원 규모 |
| 권역응급의료센터 | 9개 | 중증대응 최상위 등급 |
| 응급의료 사각지대 | 6개 시군 | 기관 0곳 (본 데이터 기준) |
| 인구 10만명당 평균 | 0.53개 | 형평성 기준선 |

## 차트
- **시군별 인구 10만명당 기관 수** (가로 막대) — 인구 대비 부족 지역이 아래쪽에 드러남. 기관 0곳은 빨강.
- **등급 구성** (도넛) — 권역 9 / 지역센터 33 / 지역기관 30.
- **이중 취약도** (산점도) — 구조 취약(x=인구10만명당) × 규모(y=총 기관수). 실시간 수용여력 축은 2단계 확장.

## 데이터 한계 (중요)
- 실시간 병상·수용가능 정보는 없음 → 구조적 분석에 한정(실시간은 2단계 API 확장 예정).
- 시군 사각지대는 **행정 지표**이며 인접 시군 접근성과 다를 수 있음.
- 인구가 적은 시군은 10만명당 수치가 과대평가될 수 있어 절대 기관 수와 함께 볼 것.
- 인구는 추정치. 공식 주민등록인구로 교체 시 정확도 상승.
